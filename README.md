# Commandline-Helper

### Install TensorFlow: 

Follow the instructions on the [official website](https://www.tensorflow.org/versions/r0.9/get_started/os_setup.html).

### Set PYTHONPATH variable:

```
export PYTHONPATH=`pwd`
```
