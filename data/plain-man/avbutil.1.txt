NAME
     avbutil -- manage AVB features and settings.

SYNOPSIS
     avbutil

DESCRIPTION
     The avbutil executable is used for the managment AVB features and set-
     tings.

     The following options are available:

     --virtual-audio enable | disable | list [if-name]
         Enable or disable the virtual audio entity on the specified inter-
         face, or list the set of interfaces with a virtual audio entity
         enabled.  An interface must be present and enabled for AVB use to
         enable the virtual audio entity on that interface. A virtual audio
         entity can always be removed from an interface regardless of if the
         interface is present or not.
