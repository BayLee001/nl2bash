NAME
       docker-swarm-join-token - Manage join tokens



SYNOPSIS
       docker swarm join-token OPTIONS ⟨worker|manager⟩



DESCRIPTION
       Manage join tokens



OPTIONS
       -q, --quiet[=false]
           Only display token


       --rotate[=false]
           Rotate join token
