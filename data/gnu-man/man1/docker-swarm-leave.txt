NAME
       docker-swarm-leave - Leave the swarm



SYNOPSIS
       docker swarm leave [OPTIONS]



DESCRIPTION
       Leave the swarm



OPTIONS
       -f, --force[=false]
           Force this node to leave the swarm, ignoring warnings

       docker-swarm(1)
