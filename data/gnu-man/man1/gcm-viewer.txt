NAME
       gcm-import - GNOME Color Manager Profile Viewer Tool

SYNOPSIS
       gcm-import [ --verbose ]


DESCRIPTION
       This manual page documents briefly the gcm-import command.

       gcm-import allows you to view properties of ICC files.
