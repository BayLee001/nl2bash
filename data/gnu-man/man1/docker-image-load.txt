NAME
       docker-image-load - Load an image from a tar archive or STDIN



SYNOPSIS
       docker image load [OPTIONS]



DESCRIPTION
       Load an image from a tar archive or STDIN



OPTIONS
       -i, --input=""
           Read from tar archive file, instead of STDIN


       -q, --quiet[=false]
           Suppress the load output



SEE ALSO
       docker-image(1)
