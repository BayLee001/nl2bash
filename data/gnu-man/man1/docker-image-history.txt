NAME
       docker-image-history - Show the history of an image



SYNOPSIS
       docker image history [OPTIONS] IMAGE



DESCRIPTION
       Show the history of an image



OPTIONS
       -H, --human[=true]
           Print sizes and dates in human readable format


       --no-trunc[=false]
           Don't truncate output


       -q, --quiet[=false]
           Only show numeric IDs
