NAME
       docker-container-kill - Kill one or more running containers



SYNOPSIS
       docker container kill [OPTIONS] CONTAINER [CONTAINER...]



DESCRIPTION
       Kill one or more running containers



OPTIONS
       -s, --signal="KILL"
           Signal to send to the container



SEE ALSO
       docker-container(1)
