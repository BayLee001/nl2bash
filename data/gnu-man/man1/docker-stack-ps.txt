NAME
       docker-stack-ps - List the tasks in the stack



SYNOPSIS
       docker stack ps [OPTIONS] STACK



DESCRIPTION
       List the tasks in the stack



OPTIONS
       -f, --filter=
           Filter output based on conditions provided


       --no-resolve[=false]
           Do not map IDs to Names


       --no-trunc[=false]
           Do not truncate output



SEE ALSO
       docker-stack(1)
