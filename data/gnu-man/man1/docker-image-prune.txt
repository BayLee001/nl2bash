NAME
       docker-image-prune - Remove unused images



SYNOPSIS
       docker image prune [OPTIONS]



DESCRIPTION
       Remove unused images



OPTIONS
       -a, --all[=false]
           Remove all unused images, not just dangling ones


       -f, --force[=false]
           Do not prompt for confirmation



SEE ALSO
       docker-image(1)
